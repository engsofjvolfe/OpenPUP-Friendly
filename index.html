OpenPUP — Protocolo Universal de Prompt

Bem-vindo ao OpenPUP!

Esta ferramenta ajuda você a criar mensagens (prompts) bem estruturadas para usar com inteligência artificial.

Basta preencher os campos abaixo, etapa por etapa, e o OpenPUP monta automaticamente um prompt que você poderá copiar e colar em seu agente de IA preferido.

Se tiver dúvidas, clique nos ícones de ajuda ao lado dos títulos — eles explicam cada parte da interface.

OpenPUP não é infalível. Estruture, Experimente, Quebre, Reconstrua.

Legenda dos Códigos:
  • M = Obrigatório
  • S = Desejável
  • A = Evitar
  • D = Uso de dados
Escala de Pesos:
  • 1.0 = prioridade máxima
  • 0.5 = desejável
  • -1.0 = deve ser evitado
  • Utilize quaisquer valores entre -1.0 e 1.0 - (0.8, -0.3...)
Exemplos:
M1: até 300 palavras (peso: 1.0)
M2: usar palavras simples (peso: 0.7)
S1: 3 exemplos práticos (peso: 0.5)
A1: jargão sem explicar (peso: -1.0)
D1: usar dados oficiais primeiro (peso: 0.8)
D1: usar dados de universidades reconhecidas após analisar dados oficiais (peso: 0.8)
